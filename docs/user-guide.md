## インストール

### exe版（推奨）
1. [Releases](https://github.com/kakikaki9821/TaskbarClock/releases)から最新の`TaskbarClock.exe`をダウンロード
2. 任意のフォルダに配置してダブルクリックで実行
3. タスクバー付近に時計が表示されます

### インストーラー版
1. `TaskbarClock-Setup.exe`をダウンロードして実行
2. インストール先を選択（デフォルト: `C:\Program Files\TaskbarClock`）
3. 「Windows起動時に自動実行」にチェックを入れると、PC起動時に自動で時計が表示されます

### Python版（開発者向け）
```bash
git clone https://github.com/kakikaki9821/TaskbarClock.git
cd TaskbarClock
pip install -r requirements.txt
python app.py
```

## 基本操作

### 時計の表示
起動すると、タスクバー付近に `HH:MM` 形式のデジタル時計が表示されます。

### 操作方法

| 操作 | 動作 |
|------|------|
| **左クリック** | アナログ時計のポップアップ表示/非表示 |
| **右クリック** | メニュー表示（サイズ変更/アラーム/タイマー/終了） |
| **ドラッグ** | 時計を好きな位置に移動 |

時計の位置はアプリ終了時に自動保存され、次回起動時に復元されます。

### アナログ時計
左クリックで表示されるアナログ時計は：
- 時針・分針・秒針がリアルタイムで回転
- マウスを時計の外に移動すると自動で閉じます
- 秒針はスムーズに回転（60fps）

## アラーム

### アラームの設定
1. 時計を右クリック → **「アラーム設定」** を選択
2. 設定ダイアログが開きます

### アラームの追加
1. **時刻**を設定（HH:MM形式）
2. **曜日**にチェック（繰り返したい曜日を選択）
3. **「追加」** ボタンをクリック

### アラームの管理
- **ON/OFF**: リスト内のアラームを選択して「ON/OFF」ボタンで切替
- **削除**: リスト内のアラームを選択して「削除」ボタン

### アラーム発動時
- Windows トースト通知が表示されます
- アラーム音が鳴ります
- **スヌーズ**: 通知の「スヌーズ (5分)」をクリックすると5分後に再通知

---

## タイマー

### タイマーの設定
1. 時計を右クリック → **「タイマー」** を選択
2. 設定ダイアログが開きます

### 使い方
- **時間入力**: 分と秒を直接入力
- **プリセット**: 1分 / 3分 / 5分 / 10分 / 30分 ボタンでワンタッチ設定
- **開始**: 「開始」ボタンでカウントダウン開始
- **一時停止/再開**: 「一時停止」ボタンで停止、再度クリックで再開
- **キャンセル**: 「キャンセル」ボタンでタイマー停止

### タイマー終了時
- Windows トースト通知が表示されます
- アラーム音が鳴ります
- ダイアログを閉じてもタイマーはバックグラウンドで継続します

---

## サイズ変更

時計を右クリック → **「サイズ」** から選択：

| サイズ | 幅x高さ | フォント | 用途 |
|--------|---------|---------|------|
| **小** | 60x24 | 10pt | 省スペース |
| **中**（デフォルト） | 80x32 | 14pt | 標準 |
| **大** | 110x40 | 18pt | 見やすさ重視 |
| **特大** | 140x50 | 24pt | 高解像度モニター向け |

サイズ設定は自動保存されます。

---

## テーマ

Windows 11 の「設定 → 個人用設定 → 色 → モードを選ぶ」で設定したテーマ（ダーク/ライト）に自動的に追従します。

- **ダークモード**: 暗い背景に白文字
- **ライトモード**: 暗い半透明背景に白文字
- テーマ切替はリアルタイムで反映されます（アプリの再起動不要）

## 設定ファイル

設定は以下の場所に自動保存されます：

```
C:\Users\<ユーザー名>\.taskbar-clock\config.json
```

### 設定項目

| キー | 型 | デフォルト | 説明 |
|------|------|-----------|------|
| `alarms` | array | `[]` | アラーム設定の一覧 |
| `theme` | string | `"auto"` | テーマ（auto/dark/light） |
| `auto_start` | bool | `false` | 自動起動 |
| `clock_size` | string | `"medium"` | 時計サイズ（small/medium/large/xlarge） |
| `clock_position` | array | `null` | 時計位置 [x, y] |
| `window_opacity` | float | `0.95` | ウィンドウ透明度 |

### ログファイル

```
C:\Users\<ユーザー名>\.taskbar-clock\logs\taskbar_clock_YYYY-MM-DD.log
```

ログは500MBでローテーション、30日間保持されます。

## トラブルシューティング

### 時計が表示されない
1. タスクバーの「隠れたインジケーター」（^マーク）をクリックして、TaskbarClockのアイコンがないか確認
2. タスクマネージャーで`TaskbarClock.exe`が動作しているか確認
3. 別のインスタンスが起動中の場合、新しいインスタンスは起動しません

### アラーム音が鳴らない
1. Windowsの音量がミュートになっていないか確認
2. `resources/alarm.wav`ファイルが存在するか確認
3. サウンドデバイスが正常に動作しているか確認
4. アプリは3段階のフォールバック（QMediaPlayer → winsound → システムビープ）を試みます

### 設定がリセットされる
`config.json`が破損した場合、自動的にデフォルト設定にリセットされます。破損したファイルは`config.json.corrupted`としてバックアップされます。

### SmartScreen警告が出る
初回実行時にWindowsのSmartScreen警告が出る場合があります。「詳細情報」→「実行」で起動できます。これはコード署名がないためです。

### 高DPIモニターで表示がおかしい
- サイズを「大」または「特大」に変更してみてください
- Windows の「表示スケール」設定を確認してください

## アンインストール

### インストーラー版
「設定 → アプリ → TaskbarClock → アンインストール」または「スタートメニュー → TaskbarClock → Uninstall」

### exe版
1. タスクバーの時計を右クリック → 「終了」
2. `TaskbarClock.exe`を削除
3. 設定を完全に削除する場合: `C:\Users\<ユーザー名>\.taskbar-clock\`フォルダを削除
